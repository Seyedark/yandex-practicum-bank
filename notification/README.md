# notification

Описание проекта

Приложение представляет сервис уведомлений клиента:

-Вычитывает сообщения из соответствующих топиков кафка

-Outbox паттерн для вычитки заранее собранных уведомлений

Приложение получает свои конфигурации из отдельного конфигурационного приложения.
В базовых настройках приложение будет доступно по http://localhost:8085/

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar notification.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t notification .", где notification имя желаемого образа
- Выполнить "docker run -p 8085:8085 notification", первый порт это порт хостовой машины, а 2 внутри docker контейнера
