# exchange

Описание проекта

Приложение представляет сервис для работы с текущими курсами валют:

-АПИ пересчёта по курсу
-АПИ для получения текущего курса валют
-Получение информации об изменении курса валют из соответствующего топика кафка

В базовых настройках приложение будет доступно по http://localhost:8087/

Приложение отправляет трейсы в zipkin, собирает метрики через prometheus и визуализирует их через grafana.

Логи приложения выводятся в консоль и отправляются в кафка, если она настроена.

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar exchange.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t exchange .", где exchange имя желаемого образа
- Выполнить "docker run -p 8087:8087 exchange", первый порт это порт хостовой машины, а 2 внутри docker контейнера
