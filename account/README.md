# account

Описание проекта

Приложение представляет сервис для работы со счетами клиента:

-АПИ для получения счёта клиента

-АПИ для получения счёта клиента и списка других пользователей

-АПИ для создания нового счёта

-АПИ для смены пароля

-АПИ для получения краткой информации о счёте

-АПИ для изменения баланса счёта

-АПИ для получения информации для перевода между счетами

-АПИ для перевода между счетами

-Outbox паттерн для вычитки заранее собранных уведомлений, сформированных при вызове определенных АПИ

Приложение получает свои конфигурации из отдельного конфигурационного приложения.
В базовых настройках приложение будет доступно по http://localhost:8085/

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar account.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t account .", где account имя желаемого образа
- Выполнить "docker run -p 8085:8085 account", первый порт это порт хостовой машины, а 2 внутри docker контейнера
