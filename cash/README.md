# cash

Описание проекта

Приложение представляет сервис внесения и снятия наличных:

-АПИ для внесения и снятия наличных
-Outbox паттерн для вычитки заранее собранных уведомлений и дальнейшая отправка в кафку

Приложение получает свои конфигурации из отдельного конфигурационного приложения.
В базовых настройках приложение будет доступно по http://localhost:8083/

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar cash.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t cash .", где cash имя желаемого образа
- Выполнить "docker run -p 8083:8083 cash", первый порт это порт хостовой машины, а 2 внутри docker контейнера
