# blocker

Описание проекта

Приложение представляет сервис блокировок:

-АПИ для получения разрешения на операцию

В базовых настройках приложение будет доступно по http://localhost:8086/

Приложение отправляет трейсы в zipkin, собирает метрики через prometheus и визуализирует их через grafana.

Логи приложения выводятся в консоль и отправляются в кафка, если она настроена.

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar blocker.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t blocker .", где blocker имя желаемого образа
- Выполнить "docker run -p 8086:8086 blocker", первый порт это порт хостовой машины, а 2 внутри docker контейнера
