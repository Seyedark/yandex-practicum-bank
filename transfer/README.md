# transfer

Описание проекта

Приложение представляет сервис переводов:

-АПИ для перевода денег с 1 счёта на другой
-Outbox паттерн для вычитки заранее собранных уведомлений и дальнейшая отправка в кафку

В базовых настройках приложение будет доступно по http://localhost:8084/

Приложение отправляет трейсы в zipkin, собирает метрики через prometheus и визуализирует их через grafana.

Логи приложения выводятся в консоль и отправляются в кафка, если она настроена.

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar transfer.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t transfer .", где transfer имя желаемого образа
- Выполнить "docker run -p 8084:8084 transfer", первый порт это порт хостовой машины, а 2 внутри docker контейнера
