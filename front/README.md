# front

Описание проекта

Приложение представляет сервис UI:

- Базовая форма login, logout, error
- Форма ввода регистрации нового аккаунта
- Форма редактирования текущего аккаунта

В базовых настройках приложение будет доступно по http://localhost:8081/

Приложение отправляет трейсы в zipkin, собирает метрики через prometheus и визуализирует их через grafana.

Логи приложения выводятся в консоль и отправляются в кафка, если она настроена.

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar front.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t front .", где front имя желаемого образа
- Выполнить "docker run -p 8081:8081 front", первый порт это порт хостовой машины, а 2 внутри docker контейнера
