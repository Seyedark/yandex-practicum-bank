# multi-bank-project

Описание проекта

Многомодульный проект банка.
Состоит из 8 приложений:

- Фронт UI(front)
- Сервис для работы со счетами клиента(account)
- Сервис уведомлений клиента(notification)
- Сервис переводов(transfer)
- Сервис внесения и снятия наличных(cash)
- Сервис блокировок(blocker)
- Сервис для работы с текущими курсами валют(exchange)
- Сервис для произвольной генерации курсов валют(generator)

Детальное описание каждого находиться в модулях.

Сборка приложения:

mvn clean install в корне главного проекта соберёт все вложенные приложения, но их можно собирать и по отдельности.

Взаимодействие между приложениями осуществляются через OAuth2.

Для корректной работы необходимо иметь настроенный keycloak и прописать необходимые конфигурации в каждом приложении

Все приложения пишут логи в консоль и в кафку, а так же приложения account, notification, transfer, cash, exchange и generator работают напрямую с кафкой

Для корректной работы необходимо иметь:
- Настроенный keycloak и прописать необходимые конфигурации в каждом приложении
- Настроенный zipkin
- Настроенный prometheus. Базовый prometheus.yml располагается по пути /prometheus
- Настроенную grafana. Базовый JSON для дашбордов (grafana.json) располагается по пути /grafana
- Настроенную кафку

Дополнительные приложения, которые можно настроить для работы с логами:
- Logstash. Необходим для сбора логов их преобразования и отправки в Elasticsearch
- Elasticsearch. Приложение хранящее логи в структурированном формате
- Kibana. Приложение для визуализации логов из Elasticsearch

Так же имеется базовые настроенных helm-чарты, которые разворачивают все микросервисы включая дополнительные, что позволяет организовать полную структуру

Запуск всех helm-чарты можно запустить с помощью helm install full-app ./umbrella-chart
