# multi-bank-project

Описание проекта

Многомодульный проект банка.
Состоит из 7 приложений:

- Фронт UI(front)
- Сервис eureka. Необходим для взаимодействия между приложениями
- Сервис config. Конфигурационный сервис, содержит настройки всех остальных
- Сервис для работы со счетами клиента(account)
- Сервис уведомлений клиента(notification)
- Сервис переводов(transfer)
- Сервис внесения и снятия наличных(cash)

Детальное описание каждого находиться в модулях.

Сборка приложения:

mvn clean install в корне главного проекта соберёт все вложенные приложения, но их можно собирать и по отдельности.

Взаимодействие между приложениями осуществляются через OAuth2.

Для корректной работы необходимо иметь настроенный keycloak и прописать необходимые конфигурации в каждом приложении
