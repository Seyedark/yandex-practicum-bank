# generator

Описание проекта

Приложение представляет сервис для произвольной генерации курсов валют и отправки их в соответствующий топик кафка:

Приложение разрабатывается как исполняемый JAR-файл

Сборка приложения:

Приложение собирается в исполняемый JAR-файл с помощью Maven:

mvn clean install

Деплой приложения:

Запустить исполняемый jar из консоли java -jar generator.jar

Для сборки приложения в docker образ нужен установленный docker и выполнение следующих шагов:

- Перейти в корень проекта
- Выполнить  "docker build -t generator .", где generator имя желаемого образа
- Выполнить "docker run -p 8088:8088 generator", первый порт это порт хостовой машины, а 2 внутри docker контейнера
